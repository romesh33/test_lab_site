TRACKS - open source приложение для управления делами. Это приложение использует подход GTD (getting things done) к тайм менеджменту. Основные идеи данного подхода:

  • Проекты - группы Задач, которые нужно выполнить.
  • Контексты - места, где задачи должны быть выполнены (дома, на работе, на конференции SQA Days).
  • Задачи (TODO) - имеют планируемую дату окончания, статус, другие параметры, которые помогают в управлении и анализе.
