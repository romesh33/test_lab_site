Итак, вы умеете создавать коллекции и запускать их через Collection Runner.

Но если вы думаете об автоматическом запуске ваших коллекций - обратите внимание на консольную утилиту Newman.

Чтобы запустить уже созданную коллекцию:
  1. Сделайте Export коллекции в json-файл.
  2. Другой вариант: получите ссылку на коллекцию через функцию Share.
  3. Запустите коллекцию через Newman в командной строке: newman run <путь_к_вашему_json>
  4. Или (если у вас есть ссылка на коллекцию): newman run <ссылка_на_коллекцию_полученная_через_Share>
  5. После успешного запуска вы можете поэкспериментировать с параметрами запуска. Например, варьировать число итераций или указывать файл с переменными для запросов.

Newman работает как NodeJS-пакет.

На ноутбуках NodeJS и Newman уже установлен.
