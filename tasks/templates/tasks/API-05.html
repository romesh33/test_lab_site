Коллекции запросов - одна из самых полезных функций Postman.

Они помогают структурировать запросы по типам или фичам, выполнять запуск наборов запросов через Runner.

Кроме того, вы можете делиться своими коллекциями с коллегами.

В рамках данного задания:
  1. Попробуйте создать коллекцию запросов. Например, вы можете назвать её "Работа с проектами" и поместить внутрь запросы по получению, созданию, изменению и удалению проекта.
  2. Запустите её через Collection Runner в Postman (Collection -> Runner).
  3. Проанализируйте результаты запуска.
