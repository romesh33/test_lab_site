Вообще-то, OWASP ZAP Proxy чаще используется для тестирования безопасности.

Но мы попробуем создать несколько юзеров при помощи функции Fuzzer.

Это будет не так просто:

  1. Сначала вам нужно настроить использование нужного прокси в браузере.
  2. Записать запрос (через прокси OWASP ZAP Proxy), который посылается при создании пользователя в интерфейсе TRACKS.
  3. Найти функцию Fuzzer в интерфейсе OWASP ZAP Proxy, выделить параметр, изменяемый при отправке запросов, и задать значения, которые вы хотите подставлять.

В качестве домашнего задания :) - попробуйте подобным образом наполнить тестовый проект тестовыми задачами.
