Вообще-то, утилита OWASP ZAP Proxy, которую мы будем использовать в этом задании, чаще применяется для тестирования безопасности.

Но мы попробуем создать при помощи неё набор TRACKS юзеров.

Предупреждаю, что это сделать непросто:

  1. Сначала нужно настроить использование прокси в браузере.
  2. Дальше - записать запрос (используя прокси OWASP ZAP Proxy), который посылается при создании пользователя в интерфейсе TRACKS.
  3. Найти функцию Fuzzer в интерфейсе OWASP ZAP Proxy
  4. Выделить параметр, изменяемый при отправке запросов, и задать значения, которые вы хотите подставлять.

В качестве дополнительного задания :) - попробуйте подобным образом наполнить TRACKS задачами.
